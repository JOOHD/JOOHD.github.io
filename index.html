-기본적인 기능으로는 카카오 회원가입, 로그인 API의 JavaScript를 사용하였습니다.

-부가적인 기능으로는 소셜커머스 댓글 기능, 책 검색 기능을 사용하여 책 정보를 불러올 수 있도록 구현하였습니다.

[프로젝트 설명]
-SDF(서울디자인재단)의 개발 요청 중에서, 새로운 카테고리의 아카이브 검색 기능 페이지입니다. 재단 API를 사용하여 Back-end 부분은 Controller의 메소드를 검색, 상세 검색, 카테고리, 상세메타 4개의 큰 부분으로 나누었습니다.

2.상단, 하단 검색 구분, 상단에는 키워드로 검색, 상세 검색 (대, 중, 소 카테고리 분류, 연도별 분류)
3.하단 카테고리 연도, 카테고리 ,태그 별 분류
4.해당 게시판 클릭 시, 게시물 상세로 이동.

-주요 기능
1.HttpURLConnection 이용한 REST API 데이터 사용
2.키워드 검색
3.ajax를 이용한 카테고리 분류, 연도 별, 검색
