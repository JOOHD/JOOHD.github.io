※스프링 MVC2
-(view Template, message/국제화, Validation, session/cookie, interceptor, filter, exeption, upload)

1.Thymleaf

3.Validation
-폼 입력시 오류가 발생하면, 고객이 입력한 데이터를 유지한 상태로 어떤 오류가 발생했는지 알려주는 검증에 대해서 설명.

-Bean Validation 애노테이션과 인터페이스를 스프링에 적용하여, @Validated를 Form/HttpMessageConverter에 적용.

4.Cookie/Session
-SessionManager 클래스를 사용하여 세션 생성, 종료, 보안 문제 처리 등, 여러 인터페이스를 제공하여 로그인 정보, 장바구니, 상품목록등, 사용자 설정등 요청을 관리합니다.
-Servlet 제공하는 세션 HttpSession을 생성하여 쿠키를 생성하고, @SessionAttribute을 이용하여 세션에 데이터를 보관한다.

5.Filter/Interceptor
-공통 관심사를 Filter(Servlet)/Interceptor(Spring)를 이용해 컨트롤러 로직에 공통부분을 효율적으로 처리 해준다.
